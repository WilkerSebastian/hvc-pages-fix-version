HV Compiler

O HVC é um interpretador que permite a compilação e execução de código em duas linguagens, HV Script e Assembly HV, voltadas para um computador hipotético.

Ele foi pensado para servir como compilador para linguagem hipotética do computador hipotético HV-2.

INSTALE

Windows

Baixar instalador do compilador
Faça download e execute o arquivo hvc.exe
Teste o compilador no terminal
Faça um teste para conferir se o HVC foi instalado corretamente hvc --help

Linux e Mac

Instale ferramentas de compilação
Para realizar a compilação do projeto é necessário ter instalado um compilador de C++, o Clang++ e o make para executar a rotina de compilação.
Clone o repositório
Com o terminal aberto, execute o seguinte comando: git clone https://github.com/Wilker-Sebastian/HV-Compiler.git
Acesse o diretório do projeto
Mude para pasta gerada cd ./HV-Compiler
Compile o código
Execute o comando make para compilar o código usando clang++ make
Adicione o HVC ao PATH do seu sistema
Este comando adicionará o diretório ./bin ao PATH do seu sistema. Isso permitirá que você execute o compilador de qualquer diretório do seu sistema echo 'export PATH="$PATH:'$(realpath ./bin)'"' >> ~/.bashrc && source ~/.bashrc
Teste o compilador no terminal
Faça um teste para conferir se o HVC foi instalado corretamente hvc --help

Aprenda

O que é HV?

Imagine um computador hipotético criado com intuito inteiramente didático, dedicado a viabilizar o ensino de programação por meio da demonstração básica de seu funcionamento... esse é o HV. Sua linguagem conta com instruções simples mas engenhosas, capazes de construir algoritmos complexos e refinados.

Em sua construção, um computador HV possui:

  • 100 gavetas
  • Calculadora, mostrador e teclado
  • Quadro Negro (EPI)
  • Porta-cartões
  • Folha de saída
  • Operador de sistema (CHICO)

A partir da abstração de conceitos sofisticados, o estudante será capaz de assimilar a base de programação, e ainda adquirir noções intuitivas de arquitetura de computadores.

Assembly HV

Trata-se de uma linguagem de programação de baixo nível, com uma sintaxe flexível que permite fácil entendimento na introdução ao universo de resolução de problemas e criação de algoritmos.

Suas instruções são sucintas, elas explicitam em suma um código de operação e o endereço de uma gaveta.

Comando Descrição Assembler
0EE Copie valor da gaveta EE (cEE) para AC AC ࢎ cEE
1EE Some cEE ao AC AC ࢎ cAC cEE
2EE some cEE ao AC AC ࢎ cAC + cEE
3EE subtraia de AC o cEE AC ࢎ cAC - cEE
4EE multiplique o cAC por cEE AC ࢎ cAC * cEE
5EE divida o cAC por cEE AC ࢎ cAC / cEE
6EE se cAC > 0, vá para EE se cAC>0, então EPI ࢎ EE
7EE leia um valor e guarda na gaveta EE leia(EE)
8EE escreva cEE no dispositivo de saída escreva(cEE)
9EE vá para cEE EPI ࢎ cEE
0-N AC recebe uma constante (truque) AC ࢎ N
000 encerra execução fim

Playground

Um playground de linguagem de programação é uma ferramenta interativa fornecida por um ambiente de desenvolvimento integrado ou plataforma online que permite aos programadores escrever e testar códigos em uma determinada linguagem de programação em tempo real.

Essa ferramenta é útil para prototipar soluções, experimentar códigos e aprender novas linguagens de programação. Oferece recursos como realce de sintaxe, sugestões de código e depuração para ajudar os desenvolvedores a escreverem códigos eficientes e livres de erros sem a necessidade de configurar um ambiente de desenvolvimento completo.
